GitHub Project Setup

Summary of the steps for creating the repository and linking it with PyCharm.

1. Account and Repository Creation

Create a GitHub account and a new repository using the following naming convention: ITB2526-FirstNameLastName.

2. Two-Factor Authentication (2FA)

It is mandatory to enable two-step authentication in GitHub's security settings.

3. Clone to PyCharm

From PyCharm, select "Get from VCS" and enter your repository URL to create the local copy.

4. README.md File

Ensure that the project contains an explanatory README.md file.

5. Add Resources and Commit

Add external files, perform the Commit, and then the Push.

6. Final Result

A view of the project successfully set up and running in GitHub.
